# 软件拦截卫士 - 快速入门指南 v2.1

## 5分钟快速上手

### 第一步：确认Python已安装

打开命令提示符（CMD），输入：
```bash
python --version

如果显示版本号（如 Python 3.9.0），说明已安装。


如果没有安装：



访问 https://www.python.org/downloads/

下载 Python 3.8+

安装时勾选 "Add Python to PATH"



第二步：下载并解压


下载本项目所有文件

解压到任意文件夹（推荐 D:\SoftwareBlocker）

确保文件结构如下：


SoftwareBlocker/
├── main.py              # 主程序（自动管理员提权）
├── requirements.txt     # 依赖列表
├── config/              # 配置文件目录（自动生成）
├── backup/              # Hosts备份目录（自动生成）
└── block_log.txt        # 拦截日志（自动生成）


第三步：启动程序

直接双击运行（自动管理员提权）：


双击 main.py

程序会自动：



获取管理员权限（弹出UAC时点"是"）

检查并安装依赖包（psutil, watchdog, pystray, pillow）

启动主程序



快速测试

测试1：安装进程秒杀（30秒）


复制任意exe文件

重命名为 360setup.exe

双击运行

预期结果：程序立即被杀死，主窗口显示"安装秒杀：360setup.exe"
也可能无法杀死因为现在安装程序大多使用通用名称


测试2：白名单保护（30秒）


复制任意exe文件

重命名为 steam.exe

双击运行

预期结果：程序正常运行，不被拦截


测试3：黑名单强制拦截（30秒）★新增


点击主界面【黑名单】按钮

添加进程：test.exe

复制任意exe文件，重命名为 test.exe

双击运行

预期结果：即使不在默认拦截列表，也被强制终止，显示"黑名单拦截：test.exe"


测试4：进程拦截（30秒）


复制任意exe文件

重命名为 360se.exe

双击运行

预期结果：主窗口显示"终止进程：360se.exe"


测试5：Hosts拦截（1分钟）


点击主界面【更新Hosts】

打开浏览器访问 2345.com

预期结果：无法访问（显示空白页或错误）


测试6：文件监控（30秒）


在下载文件夹创建 2345setup.exe

预期结果：文件被自动删除，日志显示"文件拦截"



核心功能说明

安装进程秒杀

程序会自动杀死包含以下关键词的进程：


360setup, 2345setup, LDSsetup, ludashi, 鲁大师, 
haozip, duba, jinshan, pcmgr, qqpcmgr, ksafe

特点：



检测到立即杀死，无延迟

同时删除安装文件

阻止流氓软件自动安装


白名单保护（防误杀）

以下软件绝对不会被拦截：


360急救箱系列：



360急救箱.exe, 360SysRepair.exe, 360EmergencyBox.exe

360Rs.exe, 360Repair.exe, 360SysAssist.exe


正常软件：



Steam (steam.exe), GitHub Desktop (github.exe)

火绒安全 (huorong.exe), UU加速器 (uu.exe)


添加白名单：



点击【白名单】按钮

输入进程名或域名（支持通配符 *.example.com）

点击【添加】


黑名单管理（强制拦截）★新增

功能： 自定义强制拦截的进程和域名，优先级高于白名单


适用场景：



拦截特定顽固软件

阻止特定广告域名

企业环境统一管控


使用方法：



点击【黑名单】按钮（红色警告标识）

添加进程名（如 badsoftware.exe）或域名（如 ads.bad.com）

保存后立即生效，无需重启


注意： 黑名单中的项目即使与白名单重叠也会被强制拦截



日常使用

保持后台运行（推荐）


点击【最小化到托盘】或关闭窗口

程序缩至系统托盘（右下角小图标）

左键单击托盘图标显示主窗口

持续保护，拦截记录实时更新


开机自启设置 ★新增

设置开机启动：



点击主界面右上角【设置开机启动】

提示成功后，下次开机自动运行

按钮变为【取消开机启动】


取消开机启动：



点击【取消开机启动】

从注册表中移除启动项

不影响当前运行


彻底退出软件 ★新增

注意： 最小化到托盘 ≠ 退出软件


彻底退出方法：



点击主界面右上角【退出软件】（红色按钮）

或在托盘图标右键选择【退出软件】

确认后所有监控线程停止，程序完全关闭


临时关闭：



如需暂时停止拦截，点击【退出软件】

如需长期后台保护，使用【最小化到托盘】


查看拦截记录


打开主窗口

中间列表显示所有拦截操作（时间、类型、详情）

最新记录在最上方，保留最近500条

点击【查看日志】可查看完整历史记录


备份与还原

Hosts管理：



点击【Hosts管理】按钮

立即备份：创建当前Hosts副本

双击备份：还原到该时间点

清除规则：移除所有拦截规则（不影响系统原有配置）

常见问题速查 (FAQ)

问题	解决方法
提示未检测到Python	安装Python 3.8+并勾选"Add to PATH"
无法获取管理员权限	右键main.py选择"以管理员身份运行"
Hosts更新失败	检查是否已获取管理员权限，或手动运行
进程未终止	检查进程名是否匹配，或添加到黑名单强制拦截
误拦截正常软件	添加到白名单，或检查是否在黑名单中
黑名单不生效	确认进程名拼写正确，黑名单优先级最高
托盘图标不显示	重启程序，或检查pystray是否安装成功
开机启动失效	检查杀毒软件是否拦截注册表修改
如何完全卸载	先点击【退出软件】，再删除程序文件夹，最后通过【Hosts管理】恢复Hosts
